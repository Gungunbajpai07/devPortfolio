# Portfolio Website

This repository contains a personal portfolio website for Gungun Bajpai.

## Structure

- `index.html` — main portfolio page
- `style.css` — site styling
- `script.js` — page interactions and mobile navigation
- `images/` — asset folder containing the CV image

## Notes

- The hero section includes a CTA button that opens the CV image in a new browser tab.
- Navigation links scroll to page sections such as About, Skills, Experience, Projects, Education, and Contact.

## Usage

Open `index.html` in a browser to view the portfolio.
